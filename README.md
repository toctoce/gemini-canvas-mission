# gemini-canvas-mission

Gemini Canvas 미션 결과물을 관리하는 저장소입니다.

## 목적

- Gemini Canvas로 제작한 웹앱 결과물을 기록하고 공유합니다.
- 앱별 요구사항, 배포 링크, 회고를 일관된 형식으로 남깁니다.

## 결과물

1. 단어 연습 프로그램
    - Oxford 3000 핵심 단어를 기반으로 매일 5개씩 학습하며, 사용자가 직접 작성한 예문에 대해 Gemini AI의 실시간 피드백을 받을 수 있는 실용적인 단어 학습 도구다.

2. DevType
    - 일반적인 타자 연습과 달리 실제 프로그래밍 언어의 들여쓰기와 특수문자 환경을 반영하여, 코딩에 최적화된 타자 속도와 정확도를 기를 수 있는 게임이다.

3. CS Master
    - 방대한 CS 전공 지식을 과목별 퀴즈 형식으로 풀어보며 자투리 시간을 활용해 효율적으로 학습하고, 북마크 기능을 통해 취약한 부분을 복습할 수 있는 웹 앱이다.

4. 소소한 주간 루틴
    - 요일별 또는 횟수 기반의 맞춤형 루틴을 설정하고, 친구들과 실시간 달성도를 공유하며 꾸준한 습관 형성을 돕는 커뮤니티형 루틴 관리 서비스다.
